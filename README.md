# 五子棋大师

在线五子棋训练平台，集布阵教学、AI 对战与双人对弈于一体，帮助练习者快速掌握五子棋核心战术。

## 功能

### 三大模式

| 模式 | 说明 |
|------|------|
| **双人对战** | 本地双人对弈，黑先白后，轮流落子 |
| **电脑挑战** | 三级 AI 难度，从入门到大师 |
| **布阵教学** | 12 种经典棋型预设，含战术讲解 |

### AI 难度等级

| 等级 | 算法 | 棋力 |
|------|------|------|
| **入门** | 全量位置评分择优 | 有基本攻防意识 |
| **进阶** | Minimax 3 层搜索 + α-β 剪枝 | 可预判 3 步 |
| **大师** | Minimax 7 层搜索 + α-β 剪枝 | 棋力强劲 |

### 布阵教学（12 种棋型）

**基础棋型**
- 活三 — 两端无阻挡的三连子
- 冲四 — 一端受阻的四连威胁
- 活四 — 两端无阻挡的四连必胜型

**进攻杀着**
- 双活三 — 一子形成两个活三
- 活三冲四 — 同时形成活三和冲四
- 双冲四 — 同时形成两个冲四
- 一子双杀 — 一子落下双线活三

**高级战术**
- 连续冲四 VCF — 声东击西的冲四链
- 反守为攻 — 防守中创造进攻机会
- 抓禁手取胜 — 利用三三禁手规则
- VCT 威胁链 — 连续活三威胁取胜
- 八卦阵防守 — 经典网状防御体系

### 其他特性

- 棋盘行列坐标标注（A-O / 1-15）
- 胜利全屏粒子特效
- 悔棋功能（按钮 + Ctrl+Z）
- 走棋记录实时显示
- 深色主题 UI

## 快速开始

直接用浏览器打开 `gomoku.html` 文件即可运行，无需安装任何依赖。

也可以使用任意静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/gomoku.html`。

## 操作说明

- **落子**：点击棋盘交叉点
- **悔棋**：点击「悔棋」按钮或按 `Ctrl+Z`（电脑模式自动撤回两步）
- **模式切换**：点击侧边栏「双人对战」「电脑挑战」「布阵教学」
- **难度选择**：电脑挑战模式下选择入门/进阶/大师
- **新游戏**：点击「新游戏」按钮或胜利动画中的「再来一局」
- **布阵教学**：点击棋型列表切换不同阵法，可自由点击棋盘摆子学习

## 技术实现

- 纯前端实现，单个 HTML 文件（HTML + CSS + JavaScript）
- Canvas 绘制棋盘与棋子
- AI 引擎：评估函数 + Minimax + Alpha-Beta 剪枝
- 棋盘坐标系：15×15 标准五子棋盘
